## Core Projects

### 1. [The Modulator Contrast: Predictive Leverage & Sampling Volatility](./Case%20Study:%20The%20Modulator%20Contrast.ipynb)
- **Methodologies:** Joint Multivariate Normal Distributions, Ordinary Least Squares (OLS) Regression, Cook's Distance Diagnostics, Post-Hoc Power Analysis.
- **Summary:** Built a synthetic patient cohort simulation ($N=50$) to evaluate model sensitivity in small-sample clinical trials. Proved mathematically how a single high-leverage observation ($\text{Cook's D} = 0.225$) can warp linear estimators, and utilized power diagnostics to establish optimal sample size thresholds ($N=123$) to prevent Type II errors.
